# 🔐 AI Autonomous Security Scanner

An AI-powered autonomous security testing tool that detects server-side vulnerabilities using intelligent crawling, endpoint discovery, and advanced fuzzing techniques.

---

## 🚀 Features
- 🤖 Human-like browser automation
- 🔍 Automatic API endpoint discovery
- ⚡ Multi-method fuzzing (GET, POST, PUT, PATCH, DELETE)
- 💣 Advanced payload injection
- 📊 Sensitive data detection (tokens, emails, admin flags)
- 📝 Automatic vulnerability logging

---

## 🧠 How It Works
The tool launches a real browser session and simulates human behavior to avoid detection. It crawls the application, extracts hidden API endpoints, and performs server-level fuzzing using multiple payloads. Responses are analyzed to detect sensitive data leaks and vulnerabilities.
